# Simple Calculator 🧮

## Description

A **simple calculator** is a small program that performs basic mathematical calculations.
The user enters numbers and selects a mathematical operator, and the program shows the calculated result.

This calculator focuses on the four basic arithmetic operations.

## Supported Operations

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division

## Features

* Clean and simple design
* Easy to use interface
* Basic arithmetic functionality
* Lightweight project structure

## Technologies Used

This project is built using:

* **HTML** – to create the structure of the calculator
* **CSS** – to design and style the layout

## Project Goal

The goal of this project is to demonstrate a **basic calculator interface using only HTML and CSS**, focusing on simple structure and design principles.
